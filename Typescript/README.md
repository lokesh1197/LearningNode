Install Typescript:
  - `npm install -g typescript`

Run typescript file:
  - `tsc <filename>`
  - It creates a .js file from the .ts file
  - Extension of the ts file is '.ts'

Files created:
  - `greeter.js` is the .js file created from `greeter.ts`

Documentation:
  - [https://www.typescriptlang.org/docs/home.html]
